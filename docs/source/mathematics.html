Writing Equations in documentationÂ¶

The preamble: defining macros, environments, etc. This only needs to be done once at the start of a document. All remaining equations will know about those definitions.

\def\RR{\mathbb{R}}
\def\bold#1{{\bf #1}}
\def\vectr#1{{\bf #1}}
\[\def\RR{\mathbb{R}} \def\bold#1{{\bf #1}} \def\vectr#1{{\bf #1}}\]

Using the new macros in an equation is as easy as:

{\bf v} \in \RR^n
\[{\bf v} \in \RR^n\]

Just to demonstrate that the definitions remain working past first use, simply add another equation.

\bold{K}\cdot\vectr{x} = \vectr{b}
\[\bold{K}\cdot\vectr{x} = \vectr{b}\]

Now that was easy.
